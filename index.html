使用cordova打包vue项目

发表于 2019-03-09

需要先下载cordova,gradle和android sdk,并配置好环境变量，主要难弄的就是下载sdk，通过sdk manager下载很方便。

#第一步：安装cordova
npm install -g cordova

#第二步：新建cordova项目
cordova create xxxx
cd xxxx
cordova platform add android

#第三步：将vue项目打包
npm run build
打包后将dist中的全部文件替换到到xxxx项目的www目录下

#第四步：cordova打包
cordova build android
后会生成app安装包，控制台有显示所在路径

#第五步：模拟器调试
模拟器需要安装Xposed和WebviewDebugHook，详见，安装好后chrome进入chrome://inspect/#devices即可看到devices，inspect后即可。
